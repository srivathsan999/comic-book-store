# ComicVerse - Comic Book Store HTML Template

A premium, commercial-grade HTML template designed for comic book stores, collectible shops, and pop-culture fan sites.

## Features

- **Responsive Design**: Built with Bootstrap 5 grid layout.
- **Dark Mode**: Fully supported with local storage persistence.
- **Premium Animations**: Powered by GSAP and ScrollTrigger.
- **Glassmorphism**: Modern UI aesthetic for featured elements.
- **Clean Code**: Semantic HTML5 and modular CSS3.

## File Structure

- `/assets/css/` - Stylesheets (style.css, dark.css, animations.css)
- `/assets/js/` - JavaScript files (main.js, theme-toggle.js, animations.js)
- `/assets/images/` - Image placeholders (Unsplash source URLs used in HTML)
- `*.html` - All template pages

## How to Use

1. Open `index.html` in your browser.
2. Customize the content and images as needed.
3. Replace Unsplash source URLs with your local images for production.
